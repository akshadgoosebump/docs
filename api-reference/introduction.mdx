## API Overview

The Erlin AI API provides RESTful endpoints for:

- **Content Generation** - Create articles programmatically
- **Brand Analytics** - Retrieve analytics and insights
- **Brand Management** - Manage company profiles and settings
- **Content Library** - Access and manage your content

## Base URL

```
https://api.erlin.ai/v1
```

## Authentication

All API requests require authentication using an API key:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.erlin.ai/v1/content
```

<Note>
  API keys can be generated from your account settings. Keep your API keys secure and never expose them in client-side code.
</Note>

## Request Format

The API accepts JSON-formatted requests:

```json
{
  "article_type": "blog_post",
  "topic": "Benefits of AI in Content Marketing",
  "keywords": ["AI", "content marketing", "automation"],
  "word_count": 1500
}
```

## Response Format

All responses are returned in JSON format:

```json
{
  "status": "success",
  "data": {
    "article_id": "art_123456",
    "content": "...",
    "metadata": {
      "word_count": 1523,
      "created_at": "2025-11-17T12:00:00Z"
    }
  }
}
```

## Rate Limits

API usage is subject to rate limits based on your plan:

| Plan | Requests per minute | Daily limit |
|------|---------------------|-------------|
| Free | 10 | 100 |
| Pro | 60 | 1,000 |
| Enterprise | Custom | Custom |

## Error Handling

The API uses standard HTTP status codes:

- `200` - Success
- `400` - Bad Request
- `401` - Unauthorized
- `429` - Rate Limit Exceeded
- `500` - Server Error

Example error response:

```json
{
  "status": "error",
  "error": {
    "code": "rate_limit_exceeded",
    "message": "You have exceeded your rate limit. Please try again later."
  }
}
```

## Available Endpoints

<AccordionGroup>
  <Accordion title="Content Endpoints" icon="file-lines">
    - `POST /content/generate` - Generate new content
    - `GET /content/{id}` - Retrieve content by ID

  <Accordion title="Brand Endpoints" icon="building">

  </Accordion>
</AccordionGroup>

## SDKs and Libraries

We provide official SDKs for popular languages:

- **JavaScript/TypeScript** - `npm install @erlin/sdk`
- **Python** - `pip install erlin-sdk`
- **PHP** - `composer require erlin/sdk`
- **Ruby** - `gem install erlin-sdk`

## Code Example

```javascript
const ErlinAI = require('@erlin/sdk');

const client = new ErlinAI({
  apiKey: 'your-api-key'
});

// Generate content
const article = await client.content.generate({
  type: 'blog_post',
  topic: 'AI in Content Marketing',
  keywords: ['AI', 'marketing', 'automation'],
  wordCount: 1500
});

console.log(article.content);
```
